fifawc - CLI app to track FIFA World Cup 2018

Sun, Feb 10, 2019 fifawc A small CLI app to track FIFA World Cup 2018. About fifawc is a CLI app written in Go to help developers keep track of the ongoing FIFA World Cup 2018 without leaving their terminals. The app uses football-data.org API to provide data. Installation Pre-requisite Install Go Set PATH variable Get and Install fifawc app with the command: go get -u github.com/nishchayp/fifawc To run fifawc app from across the system copy fifawc binary to directory pointed by PATH environmental variable.
